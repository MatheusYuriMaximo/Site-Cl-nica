# Tema WordPress NeuroAprender

Esta pasta contém uma cópia da landing page da Clínica Escola NeuroAprender em formato de tema WordPress.

## Arquivos principais

- `neuroaprender/`: pasta do tema.
- `neuroaprender/front-page.php`: página inicial da landing page.
- `neuroaprender/assets/`: imagens, CSS e JavaScript do site.
- `neuroaprender.zip`: arquivo pronto para enviar pelo painel do WordPress.

## Como instalar pelo painel

1. Acesse o painel do WordPress.
2. Vá em `Aparência > Temas`.
3. Clique em `Adicionar novo`.
4. Clique em `Enviar tema`.
5. Envie o arquivo `neuroaprender.zip`.
6. Clique em `Instalar agora`.
7. Depois clique em `Ativar`.

## Observação

Esta é uma cópia visual fiel da landing page atual em formato de tema. Para edição visual completa por GUI pelos sócios, o próximo passo recomendado é transformar os blocos em campos editáveis, usando WordPress com Gutenberg/ACF ou Elementor.
